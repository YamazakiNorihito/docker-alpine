- apk update
  利用可能パッケージのインデックスを更新

  - これを実行しても、インストールされたパッケージは更新されない
  - これから取ってくるパッケージの参照先を最新にする
  - Docker で使う場合、最初の方に書く

- apk upgrade
  現在インストールされているパッケージをアップグレード

  - 別のパッケージ管理システムの話ではあるが、 Docker ではなるべく使わずに、親 image に upgrade を依頼することを推奨している
